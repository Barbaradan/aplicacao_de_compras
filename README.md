## 💳 Simulador de Cartão de Crédito

### Descrição
Aplicação desenvolvida em Java que simula o uso de um cartão de crédito.
O usuário define um limite e pode realizar compras até atingir o valor disponível ou encerrar a execução.

--- 

### Funcionalidades
- Definir limite do cartão
- Realizar múltiplas compras
- Validar saldo disponível
- Exibir status da compra (aprovada/recusada)
- Encerrar operações manualmente

---

### Aprendizados
Neste projeto foram aplicados conceitos essenciais de Java e Programação Orientada a Objetos:

- Encapsulamento para proteger os atributos das classes
- Construtores para inicializar objetos com estado inicial
- Getters para acesso controlado aos dados
- Métodos para representar comportamentos (ex: validar limite, realizar compra)
- Uso de estruturas condicionais (if/else) para controle de decisões
- Uso de estruturas de repetição (while / do-while) para interação contínua com o usuário
- Entrada de dados com Scanner
